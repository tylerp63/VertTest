## Installation

Use the package manager [pip]([https://pip.pypa.io/en/stable/](https://jupyter.org/install)) to install jupyter.

```bash
pip install jupyter
```
## Run

```python
jupyter lab
```
## Open file

Use the file explorer in jupyter notebook to open the .ipynb file
